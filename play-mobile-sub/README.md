PLAY Mobile Sub
===============
A servlet which can send [PLAY events](https://github.com/play-project/play-commons/tree/master/play-commons-eventformat) to Android phones using Google Cloud Messaging (GCM). The phones must register with play-mobile-sub to propagate the GCM registration.
